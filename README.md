# 📝 Todo List App

A simple and responsive Todo List application built using HTML, CSS, and JavaScript. This app allows users to add, delete, and manage daily tasks efficiently.

---

## 🚀 Live Demo

🔗 **Live Website:** https://todo-list-afn.netlify.app/
📂 **GitHub Repository:** https://github.com/syedaafnan9-collab/TODO-LIST.git

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* 🛑 Prevent empty task input
* 💾 Stores tasks using localStorage
* 🔄 Tasks remain after page refresh
* 📱 Responsive and clean UI

---

## 🛠️ Technologies Used

* HTML
* CSS / Tailwind CSS
* JavaScript (DOM Manipulation, Event Handling)

---

## 🧠 Concepts Covered

* DOM Manipulation
* Event Listeners
* Conditional Logic
* Array Handling
* Local Storage (JSON.stringify & JSON.parse)
